# Setup
> ### git clone https://github.com/luish00/comic_xkcd

> ### `cd` comic_xkcd

> ### `yarn install`

> ### `yarn clean`

> ### `yarn start `

> ### `yarn android`

## the project directory, you can run:
Run react native server
> ### `yarn start`

Run android emulator
> ### `yarn android`

Clean java files
> ### `yarn clean`

Run eslit linter
> ### `yarn lint`
